# 🏃 Castle Run - Endless Runner Mobile Game

A 2D endless runner game developed with Unity for Android devices. Players navigate through an infinite environment, avoid obstacles, and achieve the highest score possible.

---

## 🎮 Gameplay

Castle Run is an endless runner where players must:

- Avoid obstacles and hazards
- Survive as long as possible
- Achieve high scores
- Improve reaction time and movement precision

The game's difficulty increases as the run progresses, creating a more challenging experience over time.

---

## ✨ Features

- Endless runner gameplay
- Real-time score tracking
- Dynamic obstacle challenges
- Progressive difficulty scaling
- Android APK build
- Mobile-friendly controls
- Lightweight and responsive gameplay
